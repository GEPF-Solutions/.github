# Gehrer & Pfanner

Two software engineers from Vorarlberg, Austria. We build things on the side — some commercial, some open source.

Our private services run on a single node OpenShift cluster hosted in a Proxmox VM at home. Deployments are fully GitOps — ArgoCD watches this GitHub org and syncs changes automatically. Everything is exposed securely via Cloudflare tunnels, no open ports.

## What we're working on

- **[RMCD Webapp](https://rmcd.app)** — Telemetry dashboard for road marking fleets. Live sensor data, GPS, weather, job management and efficiency statistics. `React` `C#`
- **[Highland Games Website](https://highland-games.gepf.at)** — Live scoring and matchup platform for the Pro Western Highland Games. `React` `C#`
- **[GEPF Website](https://gepf.at)** — This one. `React`
